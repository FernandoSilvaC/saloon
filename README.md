# 🌵 Estrela do Oeste - Premium Edition

![PyQt5](https://img.shields.io/badge/PyQt5-41CD52?style=for-the-badge&logo=qt&logoColor=white)
![SQLite](https://img.shields.io/badge/SQLite-003B57?style=for-the-badge&logo=sqlite&logoColor=white)

Software desenvolvido como projeto de estudo para praticar o desenvolvimento de interfaces gráficas modernas e gerenciamento de dados. O objetivo foi solucionar uma dificuldade fictícia de gerenciamento de um negócio online (um Saloon).

---

## 🚀 Funcionalidades

O **Estrela do Oeste** oferece um conjunto completo de ferramentas para gestão:

- **📋 Gerenciamento de Receitas**: Cadastro, edição e exclusão de pratos e bebidas.
- **🏗️ Calculadora de Ingredientes**: Cálculo automático da quantidade necessária de ingredientes com base no número de pratos.
- **📦 Controle de Estoque**: Monitoramento de itens disponíveis e categorias.
- **📄 Relatórios PDF**: Geração automática de relatórios de estoque e orçamentos detalhados utilizando a biblioteca FPDF.
- **🎨 Interface Premium**: Design moderno com estilo *Glassmorphism* (efeito de vidro) e tons de laranja/escuro.
- **🔍 Busca Inteligente**: Sistema de pesquisa e filtragem por categorias.

---

## 🛠️ Tecnologias Utilizadas

- **Linguagem**: [Python](https://www.python.org/)
- **Interface Gráfica**: [PyQt5](https://pypi.org/project/PyQt5/)
- **Banco de Dados**: SQLite3 (nativo do Python)
- **Geração de PDF**: [FPDF](https://pyfpdf.github.io/fpdf2/)

---

## 📦 Como Instalar e Rodar

### Pré-requisitos
Certifique-se de ter o Python instalado em sua máquina.

### Passos para execução:

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/FernandoSilvaC/saloon.git
   cd saloon
   ```

2. **Instale as dependências:**
   ```bash
   pip install PyQt5 fpdf
   ```

3. **Execute o aplicativo:**
   ```bash
   python EstrelaDoOeste.py
   ```
